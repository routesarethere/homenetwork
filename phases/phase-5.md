## Phase 5a — Dec 2023

It took almost a year from the first post to find time for more networking fun, but boy, did it get me going head first into spending so much more money and learning so many new things.

I wanted something fun to do over the winter and had seen many projects utilizing a Raspberry Pi, so I decided to pick one up — an RPi 4B. The first thing I did was install DietPi. From there it was super easy to install Pi-Hole for that sweet, sweet ad-blocking. I also tried the AdGuard plugin on my OPNsense box, but preferred Pi-Hole's interface. I remember having that feeling of "wow, that was almost too easy" after getting it set up.

---

## Phase 5b — Dec 2023

I kept seeing posts of people's various dashboards and decided I'd dig in. Home Assistant stood out immediately with its vast number of built-in integrations and community development and support. I also kept reading about Docker, so I decided to give both things a shot at the same time.

In no time I had Home Assistant up and running in Docker. I was like a kid in a candy store linking all my smart devices up. But... I kept seeing people recommending against running Home Assistant off an SD card.
